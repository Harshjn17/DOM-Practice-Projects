# DOM-Practice-Projects

JavaScript DOM Projects

A collection of beginner to intermediate JavaScript DOM projects built while learning front-end development.

About

This repository contains projects focused on mastering the Document Object Model (DOM) using HTML, CSS, and JavaScript. Each project is designed to improve problem-solving, DOM manipulation, event handling, and overall JavaScript skills through hands-on practice.

Tech Stack

- HTML5
- CSS3
- JavaScript (ES6+)

Projects

- Counter App
- Background Color Changer
- Random Quote Generator
- (More projects will be added regularly.)

Concepts Covered

- DOM Selection
- Event Listeners
- DOM Manipulation
- Functions
- Arrays & Objects
- Conditional Statements
- Loops
- Random Number Generation
- Template Literals
- Clipboard API
- Local Storage
- Responsive UI

Goal

The goal of this repository is to strengthen JavaScript fundamentals by building practical projects before moving on to React and full-stack development.

Repository Structure

Each project is organized in its own folder and contains:

- "index.html"
- "style.css"
- "script.js"
