🎯 Random Quote Generator

A simple and responsive Random Quote Generator built with HTML, CSS, and JavaScript. Click the New Quote button to display a randomly selected motivational quote, and use the Copy button to copy the current quote to your clipboard.

🚀 Features

- 🎲 Displays a random quote on button click
- 📋 Copy quote to clipboard
- 📱 Responsive design for mobile and desktop
- ⚡ Fast and lightweight
- 🧠 Built using JavaScript arrays, objects, DOM manipulation, and event listeners

🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

📂 Project Structure

Random-Quote-Generator/
│── index.html
│── style.css
│── script.js

📚 What I Learned

- Working with arrays of objects
- Generating random values
- Selecting and updating DOM elements
- Handling button click events
- Using the Clipboard API
- Writing reusable functions
- Building responsive layouts with CSS

▶️ How to Run

1. Clone or download the repository.
2. Open "index.html" in your browser.
3. Click New Quote to generate a random quote.
4. Click Copy to copy the displayed quote.
